# tello-drone

# ide
styre dronen med et gyroscope og en stick
![alt text](https://github.com/MikkelHyldgaard/tello-drone/blob/main/genialt.png?raw=true)

en gyro bruges til at styre x akse og y akse bevægelse, imen en thumbstick bruges til z aksen og rotation

# 3 lagsmodellen

præsentations-laget: bestående af dronen og de bevægler den laver
logik-laget: bestående af omregninger af data fra inputs til commands til dronen, og at sende de kommandoer trådløst
data-laget: bestående af rå inputdata som bliver behandlet i logik-laget.
